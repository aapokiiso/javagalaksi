Taustaa

Java kehitettiin 1990-luvun alussa Sun Microsystemsin toimesta, ja sen pääkehittäjät olivat Bill Joy ja James Gosling kollegoineen. Javan kehittämisen päätavoite oli alusta alkaen tuottaa helppokäyttöinen, sopeutuva ja kehittäjäystävällinen kieli. Sen eduksi nousi alusta alkaen mahdollisuus luoda Internettiin appletteja(sovelmia), joiden avulla se nousikin heti ohjelmistopiirien puheenaiheeksi. Myöhemmin siihen lisättiin lisää grafiikkapohjaisia luokkia, ja nykyään se on yksi maailman suosituimmista ohjelmointikielistä ja Java onkin käytössä lähes neljässä miljardissa laitteessa.

Yleistä

Java on oliopohjainen kieli, mikä tarkoittaa sitä, että ohjelmat koostuvat osista, "olioista", jotka pystyvät käsittelemään tietoa, vastaanottamaan sitä ja lähettämään sitä muille olioille. Käytännössä oliot voidaan ajatella soluina, jotka ovat yhteydessä toisiinsa.

Java on myös siitä käytännöllinen, että sillä tehdyt ohjelmat suoritetaan virtuaalikoneissa, toisin kuin perinteisesti suoraan järjestelmästä. Tämä mahdollistaa ohjelmien ajamisen monilta eri alustoilta, kännyköistä lentokoneisiin.
