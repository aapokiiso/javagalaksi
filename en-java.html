About Java

Java was developed in the beginning of the 1990s by the folks at Sun Microsystems, and its head developers were Bill Joy and James Gosling(and of course their colleagues). The main reason to develop Java was all the way from the beginning to create an easy, adjustable and developer-friendly language. When it was then released, people recognized right away its benefits to be especially the ability to create applets to the Internet, and Java soon became a major and widely used programming language. Later, the people at Sun added more graphics-based classes & other neat things, and nowadays Java is one of the most common programming languages, used almost in 4 billion gadgets.

General

Java is an object-oriented language, what means that the programs consists of particles called "objects", which can receive data, handle it and send it to other objects. Practically objects can be thought as cells, which are connected to each other

Java is also practical in many ways, like you dont have to run your programs straight from the system(which would be much harder), but you can use a virtual machine, that translates your Java code to machine language. This enables the programmer to run her/his programs easily from many platforms, ranging from phones to airplanes.
