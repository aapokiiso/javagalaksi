Welcome to JavaGalaksi!

JavaGalaksi is a finnish, private release channel for small programs, that usually have been made with Java. In the future, we will probably add programs made with other languages, like Python, Action Script or C++.

The site is clearly made as a hobby, and the admin is a pure amateur. That means there won't be any high-end projects, but more like small practice programs.

These little programs include several calculators, text-based programs, grapichs tests and tiny/small games.

The site features also some culture material, for example bonsai and other Japanese subjects. And the "Else"-section has... well, some random stuff.

The website is made with XHTML, also using CSS for styling, and if you have an improvement suggestion, you can send it via email to me at the Feedback-section.
